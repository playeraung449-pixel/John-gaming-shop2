အသုံးပြုပုံ လမ်းညွှန်

John Gaming Shop ကို မှန်ကန်စွာ အသုံးပြုနည်း

၁
📝 အကောင့်ဖွင့်ခြင်းနှင့် ဝင်ရောက်ခြင်း
  • ➡️ ပထမဆုံး Sign Up မှတဆင့် အကောင့်သစ်ဖွင့်ပါ။
  • ➡️ အကောင့်ဖွင့်ပြီးပါက Login ဖြင့် ဝင်ရောက်ပါ။
  • ➡️ အကောင့်ဝင်ပြီးပါက ဝန်ဆောင်မှုများကို စတင်အသုံးပြုနိုင်ပါပြီ။
၂
💰 ငွေဖြည့်သွင်းခြင်း
  • ➡️ ငွေဖြည့်ရန် ခလုတ်ကို နှိပ်ပါ။
  • ➡️ သတိပေးချက်များကို ဖတ်ပြီး ဟုတ်ကဲ့ ဆက်မည် ကိုနှိပ်ပါ။
  • ➡️ ပေးထားသော ဘဏ်အကောင့် သို့မဟုတ် Wave Pay သို့ ငွေလွှဲပါ။
  • ➡️ ငွေလွှဲပြီးပါက Slip ဓာတ်ပုံရိုက်၍ တင်ပေးပါ။
  • ➡️ Admin မှ အတည်ပြုပြီးပါက ငွေလက်ကျန်ထဲသို့ ထည့်ပေးပါမည်။
၃
🎮 ဂိမ်းပစ္စည်းများ ဝယ်ယူခြင်း
  • ➡️ ဂိမ်းများ မှ လိုချင်သော ဂိမ်းကို ရွေးပါ။
  • ➡️ ဝယ်ယူလိုသော ပစ္စည်းအမျိုးအစားကို ရွေးပါ။
  • ➡️ သင့်ရဲ့ ဂိမ်းအိုင်ဒီ (Game ID) နှင့် ဆာဗာ (Server ID) ကို ထည့်ပါ။
  • ➡️ အကောင့်စစ်မည် ကိုနှိပ်ပြီး အကောင့်မှန်ကန်ကြောင်း စစ်ဆေးပါ။
  • ➡️ ဝယ်ယူမည် ကိုနှိပ်ပါက ငွေလက်ကျန်မှ ကျသင့်ငွေ လျော့သွားမည်။
  • ➡️ ဝယ်ယူမှုမှတ်တမ်းကို မှတ်တမ်း မှ ကြည့်ရှုနိုင်ပါသည်။
၄
🎡 Spin Wheel (အခမဲ့ဆုလာဘ်)
  • ➡️ ဂိမ်းတစ်ခုဝယ်ယူပါက Spin Token 1 ခု ရရှိမည်။
  • ➡️ Spin Wheel စာမျက်နှာတွင် SPIN ခလုတ်ကိုနှိပ်ပါ။
  • ➡️ ရရှိသောဆုများ (50ks / 2 Spins / လျော့ဈေး / 100ks / Starlight Card)။
  • ➡️ Starlight Card ရရှိပါက နောင်တွင် အခြားဆုများနှင့် လဲလှယ်နိုင်ပါသည်။
၅
📜 မှတ်တမ်းနှင့် အခြေအနေ
  • ➡️ မှတ်တမ်း မှ ဝယ်ယူမှုများ၏ အခြေအနေကို ကြည့်ရှုနိုင်ပါသည်။
  • ➡️ Pending = စောင့်ဆိုင်းဆဲ (Admin အတည်ပြုရန်)
  • ➡️ Approved = အတည်ပြုပြီး (ပစ္စည်းအပ်နှံပြီး)
  • ➡️ Rejected = ပယ်ဖျက်ပြီး (ငွေပြန်အပ်မည်)
၆
🔐 စကားဝှက်ပြောင်းလဲခြင်း
  • ➡️ Menu → Password Change ကိုသွားပါ။
  • ➡️ လက်ရှိစကားဝှက်၊ စကားဝှက်အသစ်၊ အတည်ပြုရန် ထည့်ပါ။
  • ➡️ စကားဝှက်ပြောင်းမည် ကိုနှိပ်ပါ။

⚠️ သတိပေးချက် - လိမ်လည်မှုများ၊ မှားယွင်းသော အချက်အလက်များဖြင့် ဝယ်ယူပါက အကောင့် Ban ခံရနိုင်ပါသည်။

💬 ပြဿနာတစ်စုံတစ်ရာရှိပါက Admin နဲ့ဆက်သွယ်ရန် မှတစ်ဆင့် ဆက်သွယ်နိုင်ပါသည်။
