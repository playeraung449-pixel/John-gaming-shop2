1
📝 အကောင့်ဖွင့် & ဝင်ရောက်ခြင်း
▼
1
Sign Up ကိုနှိပ်ပြီး နာမည်၊ ဖုန်း၊ Email၊ Password ထည့်ကာ အကောင့်ဖွင့်ပါ။
2
အကောင့်ဖွင့်ပြီးပါက Login ဖြင့် ဝင်ရောက်ပါ။
3
ဝင်ရောက်ပြီးပါက Dashboard တွင် သင့် UID နှင့် လက်ကျန်ငွေ ကို မြင်ရပါမည်။
4
UID ကို 📋 Copy နှိပ်ပြီး မိမိလိုအပ်သလို သိမ်းဆည်းနိုင်ပါသည်။
2
💰 ငွေဖြည့်သွင်းခြင်း
▼
1
Dashboard ရှိ 💳 ငွေဖြည့်မည် (သို့) Menu ထဲမှ ငွေဖြည့်ရန် ကိုနှိပ်ပါ။
2
သတိပေးချက်များကို ဖတ်ပြီး ဆက်လက်ပြုလုပ်မည် ကိုနှိပ်ပါ။
3
ပြသထားသော KBZPay / WavePay အကောင့်သို့ ငွေလွှဲပါ။
4
ငွေလွှဲပြီးပါက Slip ဓာတ်ပုံ ရိုက်ပြီး Upload တင်ပါ။
5
Admin မှ အတည်ပြုပြီးပါက သင့်လက်ကျန်ငွေထဲသို့ အလိုအလျောက် ထည့်ပေးပါမည်။
3
🎮 ဂိမ်းပစ္စည်းများ ဝယ်ယူခြင်း
▼
1
ဂိမ်းများ မှ လိုချင်သော ဂိမ်း (ML, PUBG, FF, etc.) ကိုရွေးပါ။
2
ဝယ်ယူလိုသော ပစ္စည်းအမျိုးအစား (ဥပမာ - 86 Diamonds) ကိုရွေးပါ။
3
မိမိ၏ Game ID နှင့် Server ID ကို မှန်ကန်စွာ ထည့်ပါ။
4
အကောင့်စစ်မည် ကိုနှိပ်ပြီး အကောင့်မှန်ကန်ကြောင်း အတည်ပြုပါ။
5
ဝယ်ယူမည် ကိုနှိပ်ပါ။ လက်ကျန်ငွေလုံလောက်ပါက အလိုအလျောက် ဖြတ်ယူပြီး ဝယ်ယူမှု ပြီးဆုံးပါမည်။
6
ဝယ်ယူမှု အခြေအနေကို 📜 မှတ်တမ်း တွင် ကြည့်ရှုနိုင်ပါသည်။
4
🎡 Spin Wheel (အခမဲ့ဆုလာဘ်)
▼
✨
ဂိမ်းတစ်ခု ဝယ်ယူတိုင်း Spin Token 1 ခု အခမဲ့ ရရှိမည်။
🎰
Spin Wheel စာမျက်နှာတွင် ခလုတ်ကိုနှိပ်ပြီး ကံစမ်းပါ။
🎁
ဆုများ - 50ကျပ် / 2 Spins / 10% လျှော့ဈေး / 100ကျပ် / ⭐ Starlight Card
⭐
Starlight Card ရပါက နောင်တွင် တန်ဖိုးမြင့်ဆုများနှင့် လဲလှယ်နိုင်ပါသည်။
5
📜 မှတ်တမ်းနှင့် အော်ဒါအခြေအနေ
▼
🟡
Pending = စောင့်ဆိုင်းဆဲ (Admin အတည်ပြုရန်)
🟢
Approved = အတည်ပြုပြီး (ပစ္စည်းအပ်နှံပြီး)
🔴
Rejected = ပယ်ဖျက်ပြီး (ငွေပြန်အပ်မည် / အကြောင်းရင်းပါရှိ)
🔍
အသေးစိတ်ကြည့်ရန် အော်ဒါကို နှိပ်ပါ။
6
🔐 စကားဝှက်ပြောင်းလဲခြင်း
▼
1
Menu → Password Change ကိုသွားပါ။
2
လက်ရှိစကားဝှက်၊ စကားဝှက်အသစ် (အနည်းဆုံး 6 လုံး)၊ အတည်ပြုရန် ထည့်ပါ။
3
ပြောင်းလဲမည် ကိုနှိပ်ပါ။
✅
အောင်မြင်ပါက အလိုအလျောက် Login ပြန်ဝင်ရပါမည်။
⚠️
သတိပေးချက်
• မှားယွင်းသော Game ID ဖြင့် ဝယ်ယူပါက ပြန်အမ်းမည် မဟုတ်ပါ။
• လိမ်လည်မှုများ ပြုလုပ်ပါက အကောင့် Permanent Ban ခံရနိုင်ပါသည်။
• မည်သည့်ပြဿနာမဆို 💬 Admin နဲ့ဆက်သွယ်ရန် မှတစ်ဆင့် ဆက်သွယ်နိုင်ပါသည်။
